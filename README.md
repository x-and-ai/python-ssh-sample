# Python SSH Sample

Use python to remotely execute ssh commands

## Prerequisite

- Python 3.9 (we haven't tested on other python versions)
- pip install -r requirements.txt

## .env template

```sh
SERVER_IP=
SERVER_DOMAIN=
SERVER_PORT=
INIT_ROOT_PASSWORD=
ROOT_PASSWORD=
USER=
USER_PASSWORD=
# Private key absolute path
KEY=
# Private key passphrase
KEY_PASSPHRASE=
# Public key absolute path
PUB_KEY=
```
